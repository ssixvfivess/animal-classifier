# Итоговый проект: классификатор группы животного по фотографии

## Тема

Интеллектуальная система, которая принимает фотографию животного и определяет его группу.

Формально все животные относятся к царству Animalia, поэтому в интерфейсе используется более точная формулировка: **группа животного**.

Поддерживаемые группы:

- млекопитающее;
- птица;
- рыба;
- рептилия;
- амфибия;
- насекомое;
- паукообразное;
- ракообразное.

## Что делает проект

1. Загружает изображения животных из папок по классам.
2. Обучает модель классификации изображений.
3. Сохраняет обученную модель в файл.
4. Запускает сайт на Streamlit.
5. Позволяет загрузить фото животного и получить предсказание.

## Как подготовить данные

Фотографии нужно разложить по папкам внутри `data/animal_photos`.

Пример:

```text
data/animal_photos/mammals/cat_01.jpg
data/animal_photos/mammals/dog_01.jpg
data/animal_photos/birds/eagle_01.jpg
data/animal_photos/reptiles/snake_01.jpg
```

Названия папок — это классы, которые модель будет предсказывать.

Можно использовать только часть классов, например:

```text
mammals/
birds/
reptiles/
insects/
```

## Установка зависимостей

В терминале из корня проекта:

```bash
pip install -r requirements.txt
```

## Обучение модели

После того как изображения разложены по папкам, запустите:

```bash
python src/train_model.py
```

После обучения появятся файлы:

```text
models/animal_group_classifier.pt
models/metrics.json
models/confusion_matrix.png
```

## Быстрое обучение только последнего слоя

Если компьютер слабый, можно обучать только финальный слой ResNet18:

```bash
python src/train_model.py --freeze_backbone --epochs 5
```

## Запуск сайта

```bash
streamlit run app/app.py
```

После запуска откроется сайт. На нём можно загрузить фото животного и получить результат классификации.

## Проверка одной картинки без сайта

```bash
python src/predict.py path/to/animal.jpg
```

## Что говорить на защите

В итоговом проекте была разработана интеллектуальная система для классификации животных по фотографии. Пользователь загружает изображение через веб-интерфейс, после чего модель определяет группу животного: например, млекопитающее, птица, рыба, рептилия или насекомое.

Для решения задачи используется сверточная нейронная сеть ResNet18. Я применяю transfer learning: беру уже обученную на изображениях базовую модель и дообучаю её под свои классы животных. Это позволяет получить рабочий результат даже на небольшом учебном датасете.

Проект состоит из двух основных частей: скрипта обучения и сайта. Скрипт `train_model.py` загружает фотографии из папок, делит данные на обучение и проверку, обучает модель, считает accuracy и macro-F1, а затем сохраняет модель. Сайт `app.py` позволяет загрузить фотографию и получить предсказание с вероятностями по классам.

## Важное замечание

Без фотографий модель обучить невозможно. Сначала нужно добавить изображения в папки `data/animal_photos`, затем запустить обучение.
