# Короткая речь для защиты итогового проекта

В итоговом проекте я сделала интеллектуальную систему для распознавания группы животного по фотографии.

Пользователь загружает изображение животного на сайт, а модель определяет, к какой группе оно относится: например, млекопитающее, птица, рыба, рептилия, амфибия, насекомое, паукообразное или ракообразное.

Сначала я подготовила структуру данных. Фотографии животных раскладываются по папкам, где название папки является классом. Например, фотографии кошек, собак и лошадей лежат в папке `mammals`, птицы — в папке `birds`, рептилии — в папке `reptiles`.

Для обучения я использовала нейронную сеть ResNet18. Это сверточная модель для работы с изображениями. Я использовала transfer learning: взяла базовую модель, которая уже умеет выделять признаки на изображениях, и дообучила её под мои классы животных.

В коде есть отдельный скрипт обучения `train_model.py`. Он загружает изображения, применяет преобразования, делит данные на обучающую и проверочную выборки, обучает модель и сохраняет её в папку `models`.

Для оценки качества считаются accuracy и macro-F1. Accuracy показывает общую долю правильных ответов, а macro-F1 удобен для сравнения качества по всем классам, особенно если классы представлены неравномерно.

Интерфейс сделан на Streamlit. В файле `app.py` загружается сохранённая модель, пользователь выбирает изображение, нажимает кнопку, и сайт показывает предсказанный класс и уверенность модели.

Итог: получилась простая интеллектуальная система с обучением модели и веб-интерфейсом. Её можно дообучать на новых фотографиях и расширять список классов животных.
